🎵 MFCC Extractor App

A Python GUI application to extract Mel-Frequency Cepstral Coefficients from audio files

🚀 Quick Start

This is a desktop Python application. To use it:

  1. Clone or download this repository
  2. Install Python dependencies: pip install -r requirements.txt
  3. Run the app: python app.py

✨ Features

  • Graphical User Interface with drag-and-drop support
  • Extract MFCC features from various audio formats (WAV, MP3, FLAC, OGG)
  • Real-time display of MFCC coefficients and audio information
  • Easy file selection via browse dialog or drag-and-drop

📋 Requirements

  • Python 3.6+
  • PyQt5 (for GUI)
  • librosa (for audio processing)
  • numpy (for numerical operations)

All dependencies are listed in requirements.txt

🎯 How to Use

After launching the app:

  1. Click "Select File" to browse for an audio file, or
  2. Drag and drop an audio file onto the drop area
  3. View the extracted MFCC information in the results area

🔧 Supported Audio Formats

  • WAV
  • MP3
  • FLAC
  • OGG
  • And other formats supported by librosa

📁 Project Structure

  • app.py - Main GUI application
  • requirements.txt - Python dependencies
  • README.md - Detailed documentation

🐛 Troubleshooting

  • Ensure all dependencies are installed
  • Make sure you have a graphical desktop environment
  • For large files, processing may take time
  • Check the console for any error messages
